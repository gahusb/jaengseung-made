# 쟁승메이드 사주 서비스 카탈로그 설계

- **작성일**: 2026-04-27
- **결정 라인**: A(반복 판매 표준화) + C(LTV 락인) / D(사주집 메인 + 인스타 보조) / B(단일 코어 + 모듈)
- **상위 전략 컨텍스트**: 메모리 `project_revenue_strategy.md` (1,000만원 누적 목표, 5트랙 구조)
- **출발 사례**: 2026-04-25 사주 AI 외주 1건 납품 (49만/61만 / quote id `9dee069f-a1b1-4613-b9eb-6bf274964f94`)

## 1. 목적

방금 1건으로 검증된 사주 AI 외주를 **반복 판매 가능한 표준 카탈로그**로 굳히고, 단발 매출이 아닌 **모듈 추가 + 정기 구독으로 LTV가 확장되는 구조**로 운영한다.

"쟁승메이드는 사주 운영자에게 이런 서비스를 판다"가 카탈로그 한 장으로 명확히 보여야 한다.

## 2. 포지셔닝

> **사주 운영을 위한 AI 솔루션 — 49만원에 시작, 운영 락인까지 풀세트로.**

서브 카피
- 7년차 백엔드 개발자가 직접 만든 검증된 만세력 + AI 분석 엔진
- 컴맹도 5영업일 안에 매출 시작
- 모듈식 구조 — 필요한 만큼만 추가

## 3. 타겟 페르소나

| 인스타·틱톡 사주 인플루언서 | 메인 | 80~140만 | 카드뉴스 자동화(M3), TTS(M6), AI 크레딧(M2) |
| 사주 앱·플랫폼 창업자 | 보조 | 200~500만 | 풀 라이선스 + 커스텀, 1~2건/년 보너스 매출 |

## 4. 차별화 (3개)

1. **검증된 정확도** — `lib/saju-calculator.ts` 만세력 검증 케이스 통과(1992-12-23 男 = 壬申/壬子/癸酉/庚申), Gemini 2.5 Pro → Flash → 2.0 Flash 모델 폴백 체인 보유
2. **운영까지 락인** — 도입 + 카드뉴스 + 챗봇 + 홈페이지 + 운영 대행을 한 곳에서
3. **컴맹 친화** — 원격 설치, 영상 가이드, 카톡 응대를 전제로 설계

## 5. 코어 패키지 — 49만원 (VAT 포함)

**포함**
- AI 사주 분석 프로그램 (영구 사용권)
- 원격 설치 1회 (AnyDesk 등 원격 접속 1:1)
- 영상 가이드 5편 + PDF 매뉴얼
- 첫 1개월 카톡 응대 (무상)

**제외**
- Gemini API 사용료 (고객 부담, 월 1~2만 수준)
- 두 번째 환경 추가 설치 (15만/회)
- 호스팅 비용 (자체 사이트 구축 시)

**납기**: 착수 후 5영업일

## 8. 결제·약관 표준

| 항목 | 표준값 |

| 환불 | 착수 전 100% / 착수 후 진행률 차감 / 납품 완료 후 환불 불가 |

## 9. 운영 흐름

```
문의 (카톡·홈페이지·DM)
   ↓
HR 첫 응대 → 페르소나 판별 (사주집 / 인스타 / 앱창업)
   ↓
PM 견적서 작성 → /admin/quotes 등록 → 고객용 링크 발송
   ↓
계약금 50% 입금 → 작업 시작 (5영업일~)
   ↓
원격 설치 → 가이드 전달 → 잔금 50% 입금
   ↓
1개월 카톡 응대 (무상)
   ↓
1개월 차 후속 영업 (M9 운영 대행 / M10·M11 유지보수 제안)
   ↓
LTV 확장 (모듈 추가 / 정기 구독 전환)
```
